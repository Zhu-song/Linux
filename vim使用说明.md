# vim命令总结
## 一、模式切换

| 命令  | 作用                                |
| --- | --------------------------------- |
| i   | 在光标**前**进入插入模式                    |
| a   | 在光标**后**进入插入模式                    |
| o   | 在光标下一行新建空行并进入插入模式                 |
| 0   | 退出插入模式，回到普通模式                     |
| Esc | 退出插入模式，回到普通模式                     |
| :   | 普通模式下按 `:` 进入命令行模式（执行保存、退出、替换等操作） |
## 二、基础导航（普通模式）

| 命令      | 作用                       |
| ------- | ------------------------ |
| h/j/k/l | 左 / 下 / 上 / 右移动光标（替代方向键） |
| 0       | 跳到行首（第一个字符）              |
| $       | 跳到行尾                     |
| ^       | 跳到行首第一个非空字符              |
| gg      | 跳到文件开头                   |
| G       | 跳到指定行（如 `10G` 跳到第 10 行）  |
| 数字+G    | 跳到指定行（如 `10G` 跳到第 10 行）  |
| Ctrl+f  | 向下翻页（forward）            |
| Ctrl+b  | 向上翻页（backward）           |
| w       | 跳到下一个单词开头                |
| e       | 跳到当前单词结尾                 |
| b       | 跳到上一个单词开头                |
## 三、编辑操作（普通模式）
### 1. 删除 / 剪切

| 命令    | 作用                        |
| ----- | ------------------------- |
| x     | 删除光标所在字符（剪切到剪贴板）          |
| dd    | 删除整行（剪切到剪贴板）              |
| 数字+dd | 删除多行（如 `3dd` 删除光标开始的 3 行） |
| d$    | 删除光标到行尾的内容                |
| d0    | 删除光标到行首的内容                |
| dw    | 删除光标到下一个单词开头              |
### 2. 复制 / 粘贴

| 命令    | 作用                   |
| ----- | -------------------- |
| yy    | 复制整行                 |
| 数字+yy | 复制多行（如 `5yy` 复制 5 行） |
| p     | 粘贴到光标下一行（小写 p）       |
| P     | 粘贴到光标上一行（大写 P）       |
### 3. 撤销 / 重做

| 命令     | 作用      |
| ------ | ------- |
| u      | 撤销上一步操作 |
| Ctrl+r | 重做撤销的操作 |
### 4. 替换

| 命令  | 作用                         |
| --- | -------------------------- |
| r   | 替换光标所在字符（如 `ra` 把光标字符换成 a） |
| R   | 进入替换模式（持续替换，按 Esc 退出）      |
## 四、保存与退出（命令行模式）

| 命令           | 作用                      |
| ------------ | ----------------------- |
| :w           | 保存文件（write）             |
| :q           | 退出 Vim（quit，未修改时可用）     |
| :q!          | 强制退出，放弃所有修改             |
| `:wq` / `:x` | 保存并退出（两者等价）             |
| :w 文件名       | 另存为指定文件（如 `:w new.txt`） |
## 五、搜索与替换（高频！）
### 1. 搜索（普通模式）

| 命令    | 作用                          |
| ----- | --------------------------- |
| /关键词  | 向下搜索关键词（如 `/error` 找 error） |
| ?关键词  | 向上搜索关键词                     |
| n     | 跳到下一个匹配项                    |
| N     | 跳到上一个匹配项                    |
| :nohl | 取消搜索高亮（no highlight）        |
### 2. 替换（命令行模式）
语法：`:[范围]s/旧内容/新内容/[标志]`

| 命令               | 作用                          |
| ---------------- | --------------------------- |
| :s/old/new       | 替换当前行第一个 old 为 new          |
| :s/old/new/g     | 替换当前行所有 old 为 new（g=global） |
| :%s/old/new/g    | 替换整个文件所有 old 为 new（%= 全文）   |
| :%s/old/new/gc   | 替换时确认（c=confirm），按 y/n 选择   |
| :1,10s/old/new/g | 替换 1-10 行的所有 old 为 new      |
## 六、进阶实用命令

| 命令                   | 作用                          |
| -------------------- | --------------------------- |
| v                    | 进入可视模式（选中文本，配合 y/d 复制 / 删除） |
| V                    | 可视行模式（选中整行）                 |
| Ctrl+v               | 可视块模式（列选，适合批量编辑列）           |
| :set nu              | 显示行号（number）                |
| :set nonu            | 隐藏行号                        |
| :sp 文件名              | 水平分屏打开文件                    |
| :vsp 文件名             | 垂直分屏打开文件                    |
| `Ctrl+w` + `h/j/k/l` | 在分屏间切换光标                    |
| :r 文件名               | 读取另一个文件内容，插入到光标位置           |
| :!命令                 | 执行系统命令（如 `:!ls` 查看当前目录文件）   |
