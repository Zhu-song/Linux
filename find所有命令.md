## 一、万能结构
find 路径[排除目录] [筛选条件] [执行动作] 
## 二、筛选参数（必背）
-type f 普通文件 
-type d 目录 
-name "*.后缀" 匹配后缀 
-name "^前缀*" 匹配前缀 
-mtime +N N 天前修改 
-mtime -N N 天内修改 
-size +NM 大于 N M 
-user 用户名 所有者 
-perm 777 权限 
## 三、执行动作（必考格式）
-exec 命令 {} \; 
三条铁律： 
1. {} 前后必须有空格 
2. 结尾必须 \;
3. 命令写完整 
常用：
ls -l 看详情 
chmod 755 改权限 
rm -i 交互式删除 
mv {} 目录/ 移动
## 四、排除目录 
单目录：
find 路径 -path 目录 -prune -o 条件 -print 
多目录：
find 路径 \( -path 目录1 -o -path 目录2 \) -prune -o 条件 -print
注意：
括号前后必须空格
路径统一（全相对 / 全绝对）
## 五、企业面试铁律：分步操作 
修改 / 删除 / 移动必须两步：
1. 先查看（只筛选）
2. 再执行（加 -exec）
## 六、8 个致命错误 
1. find → fine 2.
2. 漏 -：type f → -type f 
3. {} 前后无空格 
4. 结尾少 \; 
5. 排除目录括号没空格 
6. 绝对/相对路径混用 
7. 删除不用 rm -i
8. 移动目录结尾不加 /
## 七、万能顺序口诀 
路径 → 类型 → 名字 → 时间 → 大小 → 用户 → 权限 → 排除 → 执行
